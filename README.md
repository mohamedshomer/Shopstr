Shopstr

Shopstr is a multi-store eCommerce platform designed especially for clothing stores.

The platform allows merchants to create their own online store and manage:

- Products
- Inventory
- Orders
- Customers
- Reports

Each store will have its own URL:

example:

shopstr.com/apple
shopstr.com/nike

---

Platform Components

1. Admin Panel
2. Merchant Dashboard (Flutter)
3. Storefront (Customer Store)
4. Backend API (PHP)

---

Technology Stack

Frontend:
Flutter

Backend:
PHP REST API

Database:
MariaDB / MySQL

---

Main Features

- Multi Store System
- Retail & Wholesale customers
- Inventory management
- Order management
- Reports system
- POS system
